# Project Context

This is an MVP AI clinical timeline assistant.

Goal:
Analyze therapy session transcripts and extract structured insights.

Development principles:
- Prefer simple working solutions.
- Build vertical slices.
- Avoid premature abstractions.
- Validate all LLM outputs with Pydantic.
- Challenge complexity before adding it.

Current milestone:
Single session transcript -> structured JSON insights.

Out of scope for now:
- RAG
- database
- authentication
- production deployment
